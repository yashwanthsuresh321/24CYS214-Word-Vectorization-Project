# Word Vectorizer Models Experimentation
**Course:** Machine Learning in CYS (24CYS214)  
[cite_start]**Team:** Yashwanth S (CH.SC.U4CYS24059) & Harshamithran P (CH.SC.U4CYS24021) [cite: 7]

## Overview
[cite_start]This project evaluates conventional (BoW, TF-IDF) and deep learning (Word2Vec, GloVe, BERT) vectorization techniques using the IMDb Movie Reviews dataset (50,000 samples)[cite: 57, 60].

## Performance Summary
| Method | Accuracy | F1-Score | Total Time |
| :--- | :--- | :--- | :--- |
| TF-IDF + SVM | 91.34% | 91.35% | ~13 sec |
| BERT (Fine-tuned) | 94.87% | 94.86% | ~84 min |
